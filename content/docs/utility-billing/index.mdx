title: Utility Billing & Data

---

The Utility Billing module turns monthly utility statements into structured, analyzable data. Invoices are the primary source for cost, consumption, and savings calculations across the platform.

## Ingestion

- **Arcadia (Plug)**: Primary automated path via webhooks. Arcadia-Plug-Listener receives STATUS_CHANGE / EXISTING_DATA_UPDATED / NEW_DATA_AVAILABLE events, queues via SQS, and processes into statements, accounts, meters, and charges. Full statement versioning and custom data support.
- **Urjanet**: CSV-based import utility (UrjanetImports project).
- **Manual entry**: Multiple input methods with validation.
- **Broker / direct provider feeds** and legacy rollups.

The invoice writer pipeline (adapters for Arcadia, Urjanet, manual, legacy) handles classification, FK resolution (sites, meters, service types), location backfill, and reconciliation.

## Data Model & Normalization

- Statements → accounts → meters → charges / intervals.
- Service type normalization and defaults.
- UOM handling and conversions.
- Performance Period vs. Comparison Period concepts (month-aligned or custom).
- Weather normalization and gap imputation for clean analytics.

## Validation & Quality

- Anomaly detection (IQR-based with user-adjustable sensitivity; auto-excludes bad invoices from aggregates).
- Missing data / gap tracking with banners in UI.
- Provider error surfacing and manual correction flows.
- Canonical meter resolution across data sources.

## Analytics & M&V Usage

Utility data powers:
- Performance Dashboard trends, rankings, and heat maps.
- Environmental Footprint (carbon/water derived from usage where applicable).
- Cost analytics, baseline comparisons.
- IPMVP / ASHRAE 14 M&V (in active development — see [Analytics](/docs/analytics)).

## Exports & Reporting

- Excel and PDF exports from tables.
- Structured branded PDFs for sustainability (carbon + water).
- Automated Report Cards (in progress — scheduled delivery of M&V summaries).

## Admin / Ops Tools

- Credential management (Arcadia Connect flows, MFA support).
- Account/meter assignment and batch operations.
- Webhook resend and statement discovery tools.
- Storage and cost attribution per client.

See the `HuckleberryAPI/src/lib/invoiceWriter/` and Arcadia listener for implementation details. OpenAPI endpoints under `/invoices`, `/utility/*`, and plug utility routes.
